Built a multi-stage intelligent QA system for e-commerce scenarios:
- **Intent recognition**: VLM-based user question classification and routing
- **Tool calling**: RAG retrieval, web search, and image processing with multi-round invocation
- **Retrieval pipeline**: Multimodal recall + ranking + LLM summarization
- **Model optimization**: Fine-tuning and reinforcement learning for LLM improvement
